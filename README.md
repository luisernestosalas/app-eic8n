# CRM con Clientes y Notas

Una aplicación web simple y efectiva para gestionar clientes y sus notas asociadas, desarrollada con tecnologías web estándar.

## 🚀 Características

- **Gestión de Clientes**: Crear, editar y eliminar información de clientes
- **Sistema de Notas**: Agregar notas detalladas para cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan localmente en el navegador
- **Responsive**: Compatible con dispositivos móviles y desktop

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Almacenamiento**: LocalStorage del navegador
- **Despliegue**: Vercel
