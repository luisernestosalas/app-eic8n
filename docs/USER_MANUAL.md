# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Te permite:
- Agregar nuevos clientes
- Ver la lista de todos tus clientes
- Añadir notas a cada cliente
- Buscar clientes rápidamente

## Cómo acceder

Entra a: https://app-eic8n-ib2l9eiwv-luisernestosalas-2775s-projects.vercel.app

## Funciones principales

### 1. Agregar un cliente nuevo
- Busca el botón **"Agregar Cliente"** 
- Completa los datos del cliente:
  - Nombre
  - Email
  - Teléfono
  - Empresa (opcional)
- Haz clic en **"Guardar"**

### 2. Ver tus clientes
- En la pantalla principal verás todos tus clientes
- Cada cliente muestra:
  - Nombre
  - Información de contacto
  - Número de notas que tiene

### 3. Agregar notas a un cliente
- Haz clic en el cliente que quieres
- Busca la opción **"Añadir Nota"**
- Escribe tu nota
- Guarda la nota
